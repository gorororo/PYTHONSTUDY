git 기본 명령어 배우기
git clone url . (깃허브 리펙토리 에서 주소 복사후 마지막에 공백.)
git add .
git commit -m '설명' (-m 이후에 커밋설명쓴다)
git push (깃허브로 보내기)
git pull (깃허브에서 받기)
git checkout -b '브랜치명'(깃허브 마스터로 올리기전 검수를 위해 브런치를 만들때 명령어)
git pull origin master(마스터 브랜치로부터 소스코드를 받아온다)
git push origin master (마스터 브랜치 로 보내기)
